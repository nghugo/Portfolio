SQL skills applied in this project

  • Aggregation (COUNT, SUM, GROUP BY)
  • Subquery, CTE
  • Window function and friends (LAG, RANK, PARTITION BY, ORDER BY)
  • Date manipulation (DATEPART)
  • Table handling (DROP IF EXISTS, UPDATE)

Overview

In this project, I conducted data wrangling and then exploratory data analysis on the crimes reported to the New York City Police Department (NYPD) between January 2021 and June 2021 inclusive.

For data wrangling part, I first cleaned the data by handling wrong formats, invalid values, and unnecessary prefixes. Then, I enriched the data by assigning them more meaningful values, if needed.

For data analysis part, I explored the data in 8 directions. For example, I looked at incident types, boroughs, time periods, demographics, and some relationships among the aforementioned. According to the specific investigation, I transformed the data into a suitable format, and selected among metrics such as counts, percentages and ranks.

At the end of the analysis, I concluded the findings and proposed a call to action. These may serve to reflect on and even help guide public safety, NYPDâ€™s resource management, and the government's policy design.
