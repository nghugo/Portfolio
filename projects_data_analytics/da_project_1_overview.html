EDA - London Fire Brigade - Analytics 1

Python packages applied in this project

  • General Purpose - numpy, pandas, matplotlib, seaborn
  • Interactive Map Plotting - geopandas, bokeh
  • Others - itertools, math, re

Overview

In this project, I conducted exploratory data analysis on the incident records of the London Fire Brigade (LFB) from 2009 to 2021. I explored, for example, time of call, incident type, property type, location and response time. Concurrently, I made use of the gained knowledge in cleaning, reducing and wrangling the data. The preprocessed data can then be used for modelling, perhaps in another project segment.

At the end of the analysis, I concluded the findings and proposed a call to action.

Interactive map
