Overview

Tallymeet is an app that helps users identify when to optimally schedule their next group meeting. A user first creates a poll containing various time slots for the group's consideration. The poll can then be shared by URL with group members, who indicate their availability during each time slot in three ways: "prefer", "yes", or "no". Finally, Tallymeet aggregates the responses, and ranks the time slots to maximize the number of available participants.

While the core feature of Tallymeet is to rank time slots, it is designed with flexiblity, convenience, and security in mind. For example, users do not need to register for an account to create, share, or interact with Tallymeet polls. An account simply allows a user to save polls in one place, but is not a prerequisite. In addition, after a poll is created and time slots are defined, it can be immediately shared with the group for voting. One can optionally set a poll password such that only the group of interest gets to access and interact with the poll.
