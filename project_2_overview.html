  • Others - itertools, math, scipy

Overview

The purpose of this project is to provide a prediction of expected Airbnb rental price per night in London, conditional on the metrics of a property. This not only explains to us the important factors in determining the expected rental price (and hence value) of a property, but also allows us to evaluate if a listed property is underpriced or overpriced, such that tenants and owners can both exploit this to make better decisions.

My prediction relied on regression modelling, where I considered metrics or factors such as an array of review scores, the number of reviews, the number of bedrooms and even if the host was a superhost.

In the initial parts of this project, I conducted data cleaning, train-test split and exploratory data analysis. Then, I used the gained knowledge in handling missing values and outliers, feature selection via backward elimination algorithm and manual feature selection. Finally, I ended up with a suitable regression model for price prediction, which I evaluated using diagnostic plots and metrics.

This project is a demonstration based on data in previous years. For real application, continuous market data updating is required.
